Settings Panel Popup (Haulage)

Icon-and-label toggle button panels used in the Haulage/Mining plugin. A trigger button opens a popup containing a heading and a row of selectable icon options. Selection is single-choice (radio-style). Used for operational settings like spot point distance, loading type, and cusping direction.

Anatomy

CSS Classes:

  • .mob-popup-anchor — wrapper for relative positioning
  • .mob-popup — base popup container
  • .mob-popup--settings — settings panel variant
  • .mob-popup__trigger — trigger button
  • .mob-popup__heading — panel heading text
  • .mob-popup__options — horizontal row of toggle buttons
  • .mob-popup__option — individual toggle button
  • .mob-popup__option-icon — 50×50 icon box
  • .mob-popup__option-label — label below icon
  • .is-open — show the popup
  • .is-selected — active option

XAML → CSS Mapping:

  • Placement="Center" → top: 50%; left: 100%; transform: translateY(-50%)
  • ToggleButton Style="IconToggleButton" → .mob-popup__option
  • Border 50×50 → .mob-popup__option-icon
  • TextBlock Style="HeadingMd" → .mob-popup__heading
  • StaysOpen="True" → popup stays open (demo JS handles this)

HTML Structure:

<div class="mob-popup-anchor">
  <button class="mob-popup__trigger">Trigger Label</button>
  <div class="mob-popup mob-popup--settings">
    <div class="mob-popup__heading">Panel Title</div>
    <div class="mob-popup__options">
      <button class="mob-popup__option is-selected">
        <span class="mob-popup__option-icon">icon</span>
        <span class="mob-popup__option-label">Label</span>
      </button>
      ...more options...
    </div>
  </div>
</div>
