Popup

Interactive overlay containers for tooltips, confirmation bars, suggestion lists, picker panels, inline editors, settings panels, and grouped lists. Derived from WPF <Popup> elements across Common Controls, PathBuilder, Haulage, and Vehicle Lockout plugins using AllowsTransparency="True", PopupAnimation="Fade", and BoxShadow. Dropdown overlays (IconLabelDropDownButton, ComboBoxWithButton) are covered under Forms > ComboBox.

Blazor Usage

Structure

Wrap trigger + popup in a .mob-popup-anchor for relative positioning. Toggle .is-open on the popup element to show/hide.

CSS Classes

.mob-popup — base (required on all popups)
.mob-popup--tooltip — left-placed inverted tooltip
.mob-popup--confirm — inline confirmation bar
.mob-popup--suggest — suggestion/autocomplete list
.mob-popup--picker — time/duration picker panel
.mob-popup--editor — inline editor (PathBuilder)
.mob-popup--settings — settings panel (Haulage)
.mob-popup--list — grouped list (Vehicle Lockout)
Dropdown overlays → see Forms > ComboBox

State Classes

.is-open — show popup (opacity 1, pointer-events auto)
.is-hovered — item hover state
.is-selected — item selected state
.is-pressed — item pressed state
.is-disabled — item disabled state
