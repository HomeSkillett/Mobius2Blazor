Confirmation Bar

Inline confirmation popup with close / prompt / confirm pattern. Used by ConfirmInteraction. Placement is center-right of the trigger. Buttons are TouchSize (44px) squares. Click the trigger to open; ✕ or ✓ dismisses.

Anatomy

CSS Classes

.mob-popup-anchor — wrapper for relative positioning
.mob-popup — base popup container
.mob-popup--confirm — confirmation bar variant
.mob-popup__bar — horizontal flex container
.mob-popup__btn — close/confirm button (44px square)
.mob-popup__btn--action — confirm button with BackgroundPrimary
.mob-popup__text — prompt text
.is-open — show the bar

XAML → CSS Mapping

Placement="Center" → left: 100%; top: 50%; transform: translateY(-50%)
BorderBrush="MapLabelOutlineBrush" → border-color: var(--MapLabelOutline)
Background="BackgroundStrongBrush" → background: var(--BackgroundStrong)
Width/Height="TouchSize" → width/height: var(--touch) (44px)
Margin="12,0" → padding: 0 var(--size-12)
Background="BackgroundPrimaryBrush" → background: var(--BackgroundPrimary)
