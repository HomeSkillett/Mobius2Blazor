Time / Duration Picker

Panel popup anchored below a picker input field. Used by ClockBasedTimePicker and ClockBasedDurationPicker. Click the field to toggle. Has a title bar with BackgroundPrimary and a content area for clock/input controls. Uses DropShadowEffect BlurRadius="10" → box-shadow: 0 0 10px var(--ShadowElements).

Anatomy

CSS Classes

.mob-popup-anchor — wrapper for relative positioning
.mob-popup — base popup container
.mob-popup--picker — picker panel variant
.mob-popup__title — title bar (BackgroundPrimary)
.mob-popup__body — content area for clock controls
.is-open — show the panel

XAML → CSS Mapping

Placement="Bottom" → top: 100%; left: 0
StaysOpen="False" → close on outside click
DropShadowEffect BlurRadius="10" → box-shadow: 0 0 10px var(--ShadowElements)
PopupTitle Padding="24,12" → padding: var(--size-12) var(--size-24)
PopupTitle FontSize="18" → font-size: 18px
PopupTitle Background="BackgroundPrimaryBrush" → var(--BackgroundPrimary)
ClockBasedTimePickerInput Margin="24,0" → padding: 0 var(--size-24) var(--size-24)
