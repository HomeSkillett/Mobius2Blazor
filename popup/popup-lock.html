Lock List Popup (Vehicle Lockout)

A grouped, scrollable list popup used by the Vehicle Lockout plugin to display active vehicle locks. Items are 53px-tall rows grouped under sticky headers. Supports selection, hover states, and a virtualized layout in WPF (FixedVirtualizingStackPanel). Max visible items: 8.5 rows (450.5px). Positioned via Canvas transforms in WPF; mapped to standard bottom placement in CSS.

Anatomy

CSS Classes

  • .mob-popup-anchor — wrapper for relative positioning
  • .mob-popup — base popup container
  • .mob-popup--list — list variant (320px wide, max-height 451px, overflow scroll)
  • .mob-popup__group-header — sticky group heading
  • .mob-popup__list-item — 53px item row
  • .mob-popup__list-item-icon — leading icon
  • .mob-popup__list-item-text — primary text (ellipsis overflow)
  • .mob-popup__list-item-meta — secondary text (time, status)
  • .is-open — show the popup
  • .is-selected — selected item

XAML → CSS Mapping

  • SortableGroupListView → .mob-popup--list (scrollable container)
  • GroupStyle ContainerStyle → .mob-popup__group-header
  • VehicleLockItemHeight="53" → .mob-popup__list-item { height: 53px }
  • MaxHeight="53 × 8.5" → max-height: 451px
  • FixedVirtualizingStackPanel → CSS overflow-y: auto (virtualization is runtime-only)
  • SelectionMode="Single" → .is-selected on one item at a time
  • Canvas positioning + transforms → top: 100%; left: 0
  • BoxShadow → box-shadow: 0 0 10px var(--ShadowElements)
