Suggestion List

Autocomplete suggestion list anchored below a text input. Used by SuggestionTextBox. Width matches parent input. Type to filter, click or use arrow keys + Enter to select. Mouse-over uses MouseOverPrimary at 30% opacity. Selection uses SelectionPrimary.

Anatomy

CSS Classes

.mob-popup-anchor — wrapper for relative positioning
.mob-popup — base popup container
.mob-popup--suggest — suggestion list variant
.mob-popup__suggestion — individual suggestion row
.mob-popup__suggestion-text — text label (ellipsis on overflow)
.mob-popup__suggestion-count — count badge (opacity 0.6)
.is-open — show the list
.is-hovered — MouseOverPrimary at 30% overlay
.is-selected — SelectionPrimary background

XAML → CSS Mapping

StaysOpen="False" → close on blur / outside click
Width="{ActualWidth}" → width: 100% of parent
MaximumPopupHeight → max-height: 220px; overflow-y: auto
_mouseOverElement opacity 0.3 → ::before pseudo with MouseOverPrimary
_selectionElement SelectionPrimary → background: var(--SelectionPrimary)
TextTrimming="CharacterEllipsis" → text-overflow: ellipsis
Opacity=".6" → opacity: 0.6 on count element

Keyboard Navigation

↑ / ↓ arrow keys — move selection
Enter — accept selected item
Click — accept clicked item
Blur — close list
