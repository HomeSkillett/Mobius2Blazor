Inline Editor Popup (PathBuilder)

Click-to-edit popup panels used throughout PathBuilder. A PopupToggleButton trigger opens a form popup with FieldInput controls for editing path element properties. All share: AllowsTransparency="True", PopupAnimation="Fade", StaysOpen="False", BoxShadow, and MenuAlignmentInverter for dynamic placement.

Anatomy

CSS Classes

  • .mob-popup-anchor — wrapper for relative positioning; context for MenuAlignmentInverter
  • .mob-popup — base popup container with AllowsTransparency="True" styling
  • .mob-popup--editor — inline editor variant (325px width)
  • .mob-popup--editor-narrow — 260px width variant for single-field editors
  • .mob-popup--editor-md — 300px width variant for medium-sized editors
  • .mob-popup__trigger — PopupToggleButton element
  • .mob-popup__trigger-icon — icon glyph inside trigger
  • .mob-popup__trigger-separator — "@" visual separator between value and distance
  • .mob-popup__switch — custom/default toggle row wrapping SwitchToggle
  • .mob-popup__field — field group wrapper for label + input
  • .mob-popup__field-label — field label text
  • .mob-popup__field-input — text input element bound to FieldInput
  • .mob-popup__field-row — input + unit inline row
  • .mob-popup__field-unit — unit suffix (km/h, m/s², m, etc.)
  • .is-open — active state class; applied to both .mob-popup and .mob-popup__trigger when visible

XAML → CSS Mapping

  • <PopupToggleButton> → .mob-popup__trigger; toggled by click event handler
  • <FieldInput> → .mob-popup__field + .mob-popup__field-input; bound value rendered in input
  • <SwitchToggle> → .mob-popup__switch; toggle state tracked in .demo-switch.is-on
  • <MenuAlignmentInverter> → CSS top: 100%; left: 0 on .mob-popup
  • AllowsTransparency="True" → base .mob-popup styles with transparent background
  • PopupAnimation="Fade" → transition: opacity var(--transition-fade)
  • StaysOpen="False" → close on outside click via event listener
  • BoxShadow → box-shadow: 0 0 10px var(--ShadowElements)
  • <StackPanel Width="325"> → .mob-popup--editor { width: 325px }
  • <StackPanel Width="260"> → .mob-popup--editor-narrow { width: 260px }
  • <StackPanel Width="300"> → .mob-popup--editor-md { width: 300px }

HTML Structure

<!-- PopupToggleButton trigger + popup container -->
<div class="mob-popup-anchor">
  <!-- PopupToggleButton: click toggles popup visibility -->
  <button class="mob-popup__trigger">
    <!-- Icon glyph representing element type -->
    <span class="mob-popup__trigger-icon">...</span>
    <!-- Primary value (speed, accel, distance, etc.) -->
    value text
    <!-- Separator between value and start distance -->
    <span class="mob-popup__trigger-separator">@</span>
    <!-- Start distance along path -->
    distance text
  </button>
  <!-- Popup container with AllowsTransparency, PopupAnimation=Fade, StaysOpen=False -->
  <div class="mob-popup mob-popup--editor">
    <!-- Optional: Custom/Default toggle (SwitchToggle) -->
    <div class="mob-popup__switch">...</div>
    <!-- FieldInput: Speed Limit -->
    <div class="mob-popup__field">
      <label class="mob-popup__field-label">Speed Limit</label>
      <div class="mob-popup__field-row">
        <input class="mob-popup__field-input" value="..." />
        <span class="mob-popup__field-unit">km/h</span>
      </div>
    </div>
    <!-- FieldInput: Start Distance -->
    <div class="mob-popup__field">
      <label class="mob-popup__field-label">Start Distance</label>
      <div class="mob-popup__field-row">
        <input class="mob-popup__field-input" value="..." />
        <span class="mob-popup__field-unit">m</span>
      </div>
    </div>
  </div>
</div>
