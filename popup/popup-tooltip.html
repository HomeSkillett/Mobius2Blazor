Tooltip / Details Popup

Inverted-color tooltip placed to the left of the trigger. Used by DetailsPopUp controls (e.g., warning/info symbols on SuggestionTextBox). Max width 320px, text wraps.

Interactive Demos

Anatomy

CSS Classes

.mob-popup-anchor — wrapper for relative positioning
.mob-popup — base popup container
.mob-popup--tooltip — tooltip variant (left-placed, inverted colors)
.is-open — show the tooltip

XAML → CSS Mapping

Placement="Left" → right: 100%; transform: translateY(-50%)
HorizontalOffset="11" → margin-right: var(--spacing-label-to-control)
MaxWidth="320" → max-width: 320px
Margin.All="6" → padding: var(--size-6)
BackgroundInvertPrimaryBrush → var(--BackgroundInvertPrimary)
BackgroundInvertOutlineBrush → var(--BackgroundInvertOutline)
ForegroundInvertPrimaryBrush → var(--ForegroundInvertPrimary)
PopupAnimation="Fade" → transition: opacity var(--transition-fade)

HTML Structure

<div class="mob-popup-anchor">
  <span> trigger icon </span>
  <div class="mob-popup mob-popup--tooltip">
    Tooltip message text
  </div>
</div>
