ItemsControl

Default — Vertical Stack

The base ItemsControl renders each bound item top-to-bottom in a vertical StackPanel. No selection, no scroll — just a repeater. XAML style sets BorderThickness="0" and templates a transparent Border wrapping an ItemsPresenter.

WrapPanel — Icon Grid

Empty State

When HasItems is False, a style trigger replaces the entire ControlTemplate with a placeholder message. In Blazor: conditionally render the empty message when the collection count is zero.
