List-Item Variant — Empty State

The list-item template adds a "no items" message row (row 3) that appears when the content area is empty and the expander is open. The message sits between two horizontal rules at DisabledOpacity. Source: Mobius_UI_Common_Templates_ListItemExpanderItemWithHeader.

Disabled + Collapse-Disabled

IsEnabled=False dims the entire control to DisabledOpacity (0.35) and blocks interaction. IsCollapseDisabled=True keeps the expander permanently open — the chevron is hidden and the header is non-interactive. Used when content must always be visible (e.g. required configuration sections).

IconLabelExpander

A sidebar navigation item with an icon and a horizontally collapsible label. The label slides in/out using an ExpanderItem with ExpansionDirection=ExpandLeft. ShowLabel toggles the label visibility with a 700ms ease animation. Default label width: 150px. Source: Mobius_UI_CustomControls_Templates_IconLabelExpander.
