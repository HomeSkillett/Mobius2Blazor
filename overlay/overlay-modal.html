Modal Overlay

Semi-transparent full-viewport backdrop with centered content. Used by TransparentOverlay. Backdrop uses ForegroundInvertPrimaryBrush at 40% opacity. Click outside content dismisses via CancelledCommand. Fade animation uses GeneralMouseOutDuration (500ms).

Anatomy

CSS Classes

.mob-overlay — base overlay (fixed, z-index 2000)
.mob-overlay--modal — centered modal variant
.mob-overlay__backdrop — semi-transparent click-to-dismiss layer
.mob-overlay__content — centered inner content box
.is-open — show the overlay

XAML → CSS Mapping

TransparentOverlay.BoundaryBreakingCanvas → .mob-overlay--modal
Rectangle Fill=ForegroundInvertPrimaryBrush Opacity=0.4 → .mob-overlay__backdrop
ContentControl HAlign=Center VAlign=Center → .mob-overlay__content
Active state: Grid.Opacity 0→1 → transition: opacity var(--transition-fade)
IsActivated → .is-open
CancelledCommand on Overlay_OnMouseDown → data-overlay-dismiss click handler

HTML Structure

<div class="mob-overlay mob-overlay--modal">

    Modal content here
  </div>
</div>
