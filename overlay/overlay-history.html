History Inspector Overlay (ConfigurationHistory)

Two-column slide-in inspector panel. Left column is a timeline of configuration changes grouped by time period (using ExpanderItem controls). Right column shows detailed history entries for the selected item. Separated by a 5px divider (BackgroundInvertStrongStepOneBrush). Header has a back button and title. Detail header has a SwitchToggle for Summary vs Full History view. Loading state shows ProgressRadial with purple text; empty state shows warning icon with orange text.

Anatomy

CSS Classes

.mob-overlay — base overlay (fixed, z-index 2000)
.mob-overlay--history — history inspector variant
.mob-overlay__backdrop — click-to-dismiss backdrop (25% opacity)
.mob-overlay__inspector — inspector container (720px, slides via translateX)
.mob-overlay__inspector-header — header with close + title
.mob-overlay__inspector-body — two-column body (flex)
.mob-overlay__inspector-timeline — left column (flex: 1)
.mob-overlay__inspector-divider — 5px vertical divider
.mob-overlay__inspector-detail — right column (flex: 2)
.mob-overlay__timeline-group — collapsible time group
.mob-overlay__timeline-group-header — group heading (Today, Yesterday, etc.)
.mob-overlay__timeline-item — clickable timeline entry
.mob-overlay__detail-entry — single history row
.mob-overlay__loading — purple loading indicator
.mob-overlay__empty — orange empty/warning state
.is-open — show overlay + trigger slide animation
.is-selected — highlight active timeline item
