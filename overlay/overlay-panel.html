Side Panel Overlay

Slide-in panel from the right edge. Used by NodeConfigurationOverlayControl. Slides in with 0.5s quintic easing (QuinticEase EaseOut), slides out with EaseIn. Semi-transparent backdrop at 25% opacity.

Anatomy

CSS Classes

.mob-overlay — base overlay (fixed, z-index 2000)
.mob-overlay--panel — right-edge slide-in variant
.mob-overlay__backdrop — click-to-dismiss backdrop (25% opacity)
.mob-overlay__panel — panel container (400px, slides via translateX)
.mob-overlay__panel-header — header with close + title
.mob-overlay__panel-title — heading text
.mob-overlay__panel-close — back/close button
.mob-overlay__panel-body — scrollable content area
.is-open — show overlay + trigger slide animation

XAML → CSS Mapping

NodeConfigurationOverlayControl → .mob-overlay--panel
ContentPresenter OverlayControl → .mob-overlay__panel
TranslateTransform.X: ActualWidth→0 → translateX(100%)→translateX(0)
0.5s QuinticEase EaseOut → 500ms cubic-bezier(0.23, 1, 0.32, 1)
IsOverlayOpen → .is-open
