BoxShadow

Elevation and depth tokens derived from 68 WPF DropShadowEffect usages across mobius-master. All use ShadowDepth="0" (no offset) with varying BlurRadius and opacity.

Shadow Tiers

None --shadow-none none No DropShadowEffect
Small --shadow-sm 0 0 1px var(--ShadowElements) XAML: BlurRadius="1"
Medium --shadow-md 0 0 10px var(--ShadowElements) XAML: BlurRadius="10"
Large --shadow-lg 0 0 20px var(--ShadowElements) XAML: BlurRadius="20"
Heavy --shadow-heavy 0 0 20px rgba(0,0,0,0.5) XAML: BlurRadius="20" VehicleShadowOpacity

Offset Tiers

Shadows with a y-offset for a lifted, directional feel. Used on dropdowns and dialogs that sit above content rather than float beside it.

Small Offset --shadow-sm-offset 0 2px 8px var(--ShadowElements) Combo-box dropdowns, small popovers
Medium Offset --shadow-md-offset 0 4px 12px var(--ShadowElements) Dropdown menus with subtle lift
Large Offset --shadow-lg-offset 0 4px 24px var(--ShadowElements) Dialog/modal with strong lift

Directional

Edge-anchored shadows for panels that slide in from a screen edge.

Left --shadow-left -2px 0 10px var(--ShadowElements) Right-anchored panels sliding in

Contextual Usage

How each tier maps to real component patterns in the Mobius app.

Inline Object UI

Subtle edge definition for on-object panels and inline overlays. Just enough depth to separate from the canvas.

Class: .mob-shadow-sm

Dropdown / Popover

Used for dropdown menus, error report popups, combo-box lists, and tooltip containers.

Class: .mob-shadow-md

Modal / Dialog

Floating panels, modal dialogs, video player overlays, and installer windows. Maximum separation from background.

Class: .mob-shadow-lg

Vehicle / Map Overlay

High-contrast shadow for vehicle icons and map assets that must stand out against varied map backgrounds.

Class: .mob-shadow-heavy

Dropdown Menu

Theme switcher, context menus, and combo-box dropdowns that sit below a trigger with a visible lift.

Class: .mob-shadow-md-offset

Dialog / Confirm

Confirmation dialogs and modal windows that float prominently above content with strong directional depth.

Class: .mob-shadow-lg-offset

Side Panel

Right-anchored panels (settings, history inspector) that slide in from the edge. Shadow falls left to reinforce the reveal direction.

Class: .mob-shadow-left

Blazor Usage

CSS Custom Properties (preferred)

Apply via the token directly in your component CSS: box-shadow: var(--shadow-md);
