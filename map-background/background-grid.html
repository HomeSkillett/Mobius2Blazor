Background Grid

Three-layer system: solid MapBackground fill + tiled grid with major/minor line weights + left-edge shadow overlay. Grid tile size is dynamic via GridSizeMultiConverter(param=8).

Full Composite (3 layers)

Layer 1: MapBackground solid fill (MapControl.xaml:35). Layer 2: Single-frequency tiled grid — 1px lines at 0.5 opacity with intersection emphasis at 0.75 (BackgroundGridControl.xaml:90-131). Layer 3: Left-edge shadow (MapControl.xaml:174-187). Note: the grid is intentionally very subtle — MapGridLines token at half opacity. Light theme: ~3% black. Dark/Blackout: ~3% white.

Grid Tile Anatomy

This is a SINGLE-FREQUENCY grid, not two grids at different spacings. The VisualBrush tile (4x4 WPF Grid: rows 2,*,2,1 / cols 2,*,2,1) contains 6 rectangles. Rects 1-2 draw 1px lines at tile edges (0.5 opacity). Rects 3-6 boost intersection corners to 0.75 opacity. When tiled, every crossing is slightly darker than the line runs.

Dynamic Zoom Levels

GridSizeMultiConverter subdivides the base grid (param=8) by powers of 2, capped at level 3. Tile pixel size = unitsPerPixel x (8 / 2^subdivisionLevel). Test values from GridSizeMultiConverterTests.cs. Each card shows the same SVG pattern tile structure at different sizes.

subdiv=3 (capped), gridSize=1, tile~41px
Zoomed In (1 m/px)

Medium (3 m/px)
subdiv=2, gridSize=2, tile=48px

subdiv=2, gridSize=2, tile=64px

Left-Edge Shadow

100px wide gradient overlay from ShadowElements to transparent at 0.25 opacity. Simulates sidebar shadow cast onto map. Source: MapControl.xaml lines 174-187.
