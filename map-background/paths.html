Colors used when a path is being created or edited in the PathBuilder tool. Paths render as a two-layer stroke: outer outline + inner fill, with round caps and joins. Source: PathBuilderLayerControl.xaml, LightWeightPathDrawingControl.xaml.cs

Path Builder Stroke Anatomy

Source: PathBuilderLayerControl.xaml lines 173-182, LightWeightPathDrawingControl.xaml.cs

Outer Stroke (Outline)
OutsideWidth = 12px
LineCap = Round, LineJoin = Round
Brush: PathNormalOutlineBrush (or Selected/Error)
Inner Stroke (Fill)
InsideWidth = 8px
LineCap = Round, LineJoin = Round
Brush: BackgroundStrongBrush (white)
Control Node (Circle)
CircleRadius = 6px
CircleWidth = 2px (pen thickness)
Fill: BackgroundStrongBrush, Stroke: varies by drivability
Rendering Layers
1. Outer stroke at 12px (colored outline)
2. Inner stroke at 8px (white fill)
3. Control nodes at each ControlNode point

Path Assignment States

When a path is assigned to a vehicle, it renders in one of four assignment states based on the AssignmentStatus enum (IObjective.cs). Each state has outline, fill, and connection colors with selected/not-selected variants.

Assignment State: Planning

AssignmentStatus.Planning — path is being computed by the planner. Always renders at full opacity (no selected/not-selected variant).

Planning Outline
--PathAssignment_Planning_Outline
Light purple, rgba(192, 157, 243, 1)
Planning Fill
--PathAssignment_Planning_Fill
Light purple at 46.7% opacity
Planning Connection
--PathAssignment_Planning_Connection_Outline
Darker purple for connection lines between segments

Assignment State: Planned

AssignmentStatus.Planned — plan is complete, waiting to execute. Has selected and not-selected variants.

Planned (Not Selected) Outline

--PathAssignment_Planned_NotSelected_Fill
Green at 13.3% opacity
Planned (Selected) Outline
--PathAssignment_Planned_Selected_Outline

Planned (Selected) Fill
--PathAssignment_Planned_Selected_Fill
Green at 46.7% opacity
Planned Connection (Not Selected)
--PathAssignment_Planned_Connection_NotSelected_Outline
Olive green at 33.3% opacity
Planned Connection (Selected)
--PathAssignment_Planned_Connection_Selected_Outline
Full-opacity olive green

Assignment State: Executing

AssignmentStatus.Executing — vehicle is actively driving the path. Has selected and not-selected variants.

Orange at 33.3% opacity
Executing Connection (Selected)
--PathAssignment_Executing_Connection_Selected_Outline

Assignment State: Error

AssignmentStatus.PlanFailed or ExecutionFailed — assignment has encountered an error. No selected/not-selected variant; always renders at full opacity.

Error Outline

Error Connection
--PathAssignment_Error_Connection_Outline

Path Points

Points are rendered at control nodes, action markers, endpoints, and connection points. All are circles (ellipses) drawn via DrawEllipse(). Source: AssignmentRenderingControl.xaml.cs

Point Dimensions

Point Type Radius (world units) Description Source
Control Node 6 (radius), 2 (pen) Circle at each editable node in PathBuilder. Fill = BackgroundStrong, stroke = outline color. PathBuilderLayerControl.xaml:173-174
Action Marker 2.0 Small dot at action points along an assigned path. Uses PathAction brush. CircleActionRadius — line 158
Last Point (Inner) 5.0 Filled inner circle at the assignment endpoint. LastPointInnerRadius — line 162

Enum Value Color Family Meaning
