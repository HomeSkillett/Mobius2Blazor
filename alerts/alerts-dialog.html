Confirmation Dialog

Default State

Confirm Action
Are you sure you want to remove this role assignment? This action cannot be undone.

Fixed 400×300 window. ResizeMode=NoResize. Centered on parent (CenterOwner). Closing the window fires OkCancelDialog_OnClosed. The dialog is modal — it blocks interaction with the parent window until OK or Cancel is clicked.

Interactive Demo

Click the button to open the dialog as a modal overlay with backdrop. OK, Cancel, and the X button all close it — the result is shown below.
