Dropdown Open

Click the field to open the popup dropdown. Items use TouchSize (44px) min-height. Hover shows MouseOverPrimary at 30% opacity. Selected item uses SelectionPrimary background with SelectionForeground text. Max item width is 300px. Uses div-based markup to mirror the WPF popup; the demo-only script below handles open/close (the Blazor component uses a native popup overlay).

Editable (Type-to-Select)

When IsEditable="True", a TouchTextBox overlays the content presenter, allowing free-text entry or type-to-filter. The dropdown arrow remains. Also supports IsTextSearchEnabled for jump-to-match behavior on standard (non-editable) ComboBoxes. Uses div-based markup since native <select> doesn't support editable text input. XAML: PART_EditableTextBox in the ComboBox template.

With Action Button (ComboBoxWithButton)

Extended ComboBox that adds an action button at the bottom of the dropdown popup. Used for "Add new..." or "Configure..." actions. The button sits in Grid.Row="1" with 15px horizontal margin and 10px vertical margin. The control stays enabled even when HasItems=False. XAML: ComboBoxWithButton.xaml.
