FilterBarControl

Horizontal — Default (Search Closed)

The default FilterBarControl layout. A 3-column grid: [92px toggle | auto search-expander | * category-list]. The search box starts collapsed. Toggle button uses IconToggleButton style with a 4px left selection edge. Category list is a horizontal ListBox with 12px item spacing. Source: Mobius_UI_CustomControls_Templates_FilterBar.

Horizontal — Search Open

When IsSearchBoxOpen=True, the toggle shows its 4px selection edge, and the ExpanderItem expands left to reveal a 220px TouchTextBox with 40px right margin. The input auto-focuses. Pressing Escape closes the search and clears FilterText.

Vertical — Search Closed

The vertical variant uses VerticalIconToggleButton (4px bottom selection edge instead of left, Height=40, Margin=0,0,0,-4). The search TouchTextBox uses MinWidth=220 with HorizontalAlignment=Stretch. Source: Mobius_UI_CustomControls_Templates_VerticalFilterBar.
